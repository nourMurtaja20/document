2-3 Recommended PHP Configuration Limits

White screen, demo content fails when importing, empty page content and other similar issues are all related to low PHP configuration limits. The solution is to increase the PHP limits. You can do this on your own, or contact your web host and ask them to increase those limits to a minimum as follows: 

            
        max_input_vars = 3000
        memory_limit = 128M
        max_execution_time = 300
        max_input_time = 300
        upload_max_filesize = 64M
        post_max_size = 64M

2.4 Theme Installation

Theme Installation via Wordpress:

1
Login to your Wordpress Admin Panel.

2
Go to Appearance → Themes . Click the Add New button.

3
Click on the Upload Theme button. Locate "theme.zip" and click Install Now.

4
Click on the Activate button to activate theme. Congratulations, you have now activated theme! You can now proceed with theme's Plugin Installation. image

2.5 Plug-ins Installation

All required plugins are already added in your downloaded package.

1
After theme installation you should be redirected to Install Required Plugins page, which shows list of plugins:

2
Now, Select all "plugins" Note - The installation process may take few minutes. Proceed further after successful installation.

3
When "All installations have been completed" appears, click on "Return to Required Plugins Installer".

4
Now, Select all plugins and choose Activate from drop down box this time.
